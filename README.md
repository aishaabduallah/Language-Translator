anguage Translator

A simple web app that translates basic words and sentences between English and Arabic.

Features
- Translate English to Arabic and vice versa
- Supports basic sentences
- Categorized vocabulary (Pronouns, Verbs, Nouns, etc.)

Tools Used
- Python
- Streamlit
- Pandas
