anguage Translator

A simple web application that translates basic words between English and Arabic.

Features
- Translation from English to Arabic and vice versa
- Supports basic words
- Categorized vocabulary (pronouns, verbs, nouns, etc.)

Tools Used
- Python
- Streamlit
- Pandas
